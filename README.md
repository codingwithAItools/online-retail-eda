# Online Retail Exploratory Data Analysis (EDA)

This project is part of the Google Advanced Data Analytics Professional Certificate on Coursera.

The goal of this project was to perform exploratory data analysis on transactional online retail data to uncover trends, identify top-performing products and countries, and generate business insights.

---

## Project Objectives

- Clean and prepare real-world transactional data
- Explore sales patterns and customer purchasing behaviour
- Identify top-selling products and countries
- Analyse sales trends over time
- Detect outliers and anomalies
- Generate actionable business recommendations

---

## Dataset

Dataset: Online Retail Dataset (2010–2011)

The dataset contains transactional records from a UK-based online retailer.

### Features

- InvoiceNo
- StockCode
- Description
- Quantity
- InvoiceDate
- UnitPrice
- CustomerID
- Country

---

## Tools & Libraries Used

---

## Key Insights

- November 2011 recorded the highest sales revenue
- Thursdays generated the most sales transactions
- The United Kingdom dominated total sales
- Several products generated significantly higher sales volumes than others
- The dataset contained outliers caused by bulk purchases and returns
